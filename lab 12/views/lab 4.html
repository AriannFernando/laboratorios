    Preguntas del laboratorio 4:
  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • Respuesta: Java crea aplicaciones tras complilarlas por medio de una máquina virtual, mientras que JavaScript se ejecuta de manera nativa en los navegadores al ser un lenguaje interpretado, de igual forma java es un lenguaje fuertemente tipado mientras que JavaScript es un lenguaje debilmente tipado, Java es un lenguaje orientado a objetos mientras que JavaScript es un lenguaje orientado a prototipos, Java es un lenguaje más robusto sin embargo, JavaScript ofrece más versatilidad y flexibilidad.
  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    • Respuesta: El objeto Date tiene los siguientes métodos

    • getDate(): Devuelve el día del mes, un valor numérico entre 1 y 31.

    • getDay(): Devuelve el día de la semana, un valor numérico entre 0 y 6.

    • getFullYear(): Devuelve el año con cuatro dígitos.

    • getMilliseconds(): Devuelve los milisegundos entre 0 y 9999.

    • getTime(): Devuelve los milisegundos que han transcurrido desede el día 1 de enero de 1970 y la fecha correspondiente al objeto al que se manda el mensaje.
  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    • Respuesta: Los arreglos tienen los siguientes métodos.

    • push(): Añada 1 o más elementos al final de un arreglo.

    • pop(): Elimina y devuelve el ultimo elemento del arreglo.

    • unshift(): Añade 1 o más elementos al inicio del arreglo.

    • concat(): concatena los elementos pasados por parámetro.

    • slice(init, end): Devuelve los elementos desde posición init hasta end.
  • ¿Cómo se declara una variable con alcance local dentro de una función?

    • Respuesta: Se tiene que utilizar var para declarar las variables ya que permite que la variable declarada sea actualizada y reasignada. Dicha variable tiene que ser declarada dentro de la función.
  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    • Respuesta: Las implicaciones que tiene usar variables globales dentro de funciones es que dicha variable puede ser alterada o modificada por otras funciones o procesos, al igual que dificulta la legibilidad del código, creando dependencias ocultas, siendo un código más complicado de usar.
